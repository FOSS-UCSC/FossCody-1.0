About   FOSSCODY 1.0

This is an Open source projects based competition for internal students of
University of Colombo School of Computing.

The main purpose of this competition will be
improving the skills of handling a software project using github.


  • Platform : Github
  • Competition For : 1st Year Students in UCSC
